Parameters

A unique string identifier for the target matching Tealium instance.
configuration

A TEALWKExtensionConfiguration object that contains the target account-profile-env settings.

Return Value

TEALWKExtension instance that matches a Tealium instance with the given configuration

Discussion

Warning: Note: Only the account-profile-env options will be used, all other configuration options will be ignored.
